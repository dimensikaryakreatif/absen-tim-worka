Standar Operasional Prosedur Penggunaan Alat

Aturan penanganan, perawatan, dan pertanggungjawaban aset kamera/video agensi.

Kamera & Lensa

  • Wajib pasang Strap: Tali pengaman wajib terpasang di leher/pergelangan tangan saat dioperasikan secara manual (*handheld*).
  • Ganti Lensa Aman: Posisikan bodi kamera menghadap ke bawah saat penggantian lensa untuk meminimalkan debu jatuh langsung ke sensor.
  • Penyimpanan Dry Cabinet: Bersihkan alat dari sisa sidik jari, segera masukkan kembali ke Dry Cabinet Elektrik (Kelembapan ideal 45-55% RH).

Peralatan Audio

  • Metode Over-Under: Gulung kabel mikrofon nirkabel atau kabel audio lain menggunakan metode over-under agar tembaga internal tidak patah/rusak.
  • Keluarkan Baterai Cadangan: Selalu lepas baterai AA/AAA jika perangkat audio tidak akan digunakan dalam waktu 2x24 jam untuk menghindari korosi.

Unit Drone

  • Inspeksi Fisik Baling-Baling: Selalu cek propeller sebelum terbang. Dilarang menerbangkan drone jika ada retakan sekecil apa pun.
  • Prosedur Kalibrasi: Lakukan prosedur kalibrasi GPS dan kompas di tempat terbuka sebelum meluncurkan (*take-off*).
  • Line of Sight (VLOS): Selalu pantau pergerakan drone dengan pandangan kasat mata langsung dan jangan langgar zona larangan terbang.

Lighting & Kelistrikan

  • Siklus Kipas Pendingin: Biarkan cooling fan internal berhenti bekerja secara otomatis terlebih dahulu sebelum mencabut kabel utama dari stopkontak.
  • Keamanan C-Stand: Selalu pasang kantong pasir (*sandbag*) pemberat di dasar stand lampu tinggi agar tidak goyah atau rubuh ke arah subjek syuting.

Gimbal & Aksesori

  • Penyeimbangan Akurat: Selalu jalankan proses kalibrasi beban mekanikal (*balancing*) secara manual sebelum menghidupkan motor gimbal.
  • Format Memory Dalam Kamera: Jangan pernah memformat SD card dari laptop. Selalu format langsung dari menu bodi kamera agar sistem direktori aman.

Tanggung Jawab & Sanksi

  • Kerusakan Akibat Kelalaian: Kerusakan yang terjadi akibat kecerobohan personal (jatuh, tertumpah air) sepenuhnya ditanggung oleh kru bersangkutan.
  • Penyusutan Alami: Aus fungsional perangkat karena frekuensi pemakaian sewajarnya akan ditangani secara berkala oleh manajemen agensi.

Google Sheets Real-Time Sync Setup

Agar portal absen ini dapat diakses bersama dari HP/laptop masing-masing tim dan mengalir ke satu file spreadsheet yang sama secara terpusat, tempelkan URL Google Apps Script Web App Anda di bawah ini.

Cara Setup Google Sheets Anda (Hanya 2 Menit):

  1. Buat Spreadsheet baru di Google Drive milikmu.
  2. Klik menu Ekstensi / Extensions -> pilih Apps Script.
  3. Hapus seluruh kode bawaan Google, lalu salin dan tempel (*paste*) kode di bawah ini:
function doGet(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = sheet.getDataRange().getValues();
  var rows = [];
  if (data.length <= 1) {
    return ContentService.createTextOutput(JSON.stringify([]))
      .setMimeType(ContentService.MimeType.JSON)
      .setHeader('Access-Control-Allow-Origin', '*');
  }
  for (var i = 1; i < data.length; i++) {
    rows.push({
      id: data[i][0].toString(),
      name: data[i][1],
      role: data[i][2],
      mode: data[i][3],
      division: data[i][4],
      inTime: data[i][5],
      date: data[i][6],
      outTime: data[i][7] || null,
      output: data[i][8] || ""
    });
  }
  return ContentService.createTextOutput(JSON.stringify(rows))
    .setMimeType(ContentService.MimeType.JSON)
    .setHeader('Access-Control-Allow-Origin', '*');
}

function doPost(e) {
  try {
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    var params = JSON.parse(e.postData.contents);
    var action = params.action;
    
    if (action === "checkin") {
      sheet.appendRow([
        params.id,
        params.name,
        params.role,
        params.mode,
        params.division,
        params.inTime,
        params.date,
        "",
        ""
      ]);
      return ContentService.createTextOutput(JSON.stringify({status: "success"}))
        .setMimeType(ContentService.MimeType.JSON)
        .setHeader('Access-Control-Allow-Origin', '*');
    }
    
    if (action === "checkout") {
      var data = sheet.getDataRange().getValues();
      var found = false;
      for (var i = 1; i < data.length; i++) {
        if (data[i][0].toString() === params.id.toString()) {
          sheet.getRange(i + 1, 8).setValue(params.outTime);
          sheet.getRange(i + 1, 9).setValue(params.output);
          found = true;
          break;
        }
      }
      return ContentService.createTextOutput(JSON.stringify({status: found ? "success" : "not_found"}))
        .setMimeType(ContentService.MimeType.JSON)
        .setHeader('Access-Control-Allow-Origin', '*');
    }
  } catch (err) {
    return ContentService.createTextOutput(JSON.stringify({status: "error", message: err.toString()}))
      .setMimeType(ContentService.MimeType.JSON)
      .setHeader('Access-Control-Allow-Origin', '*');
  }
}
